# TACo website

Provide insight into the workings of the TACo system.

Live at : https://tacoscan.com/
## Installation

```bash
git clone https://github.com/threshold-network/tacoscan.git
cd tacoscan
npm install -g yarn
yarn install
yarn codegen

# run in dev mode
yarn start

# run in produce mode
yarn build-prod
yarn preview

# http://localhost:4401
```
